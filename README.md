# SmartFitao Portfolio Hub

React + TypeScript showcase for all deployed links.

## Run locally

```powershell
cd "E:\fyp whole backend\Portfolio of Overall Porject"
npm install
npm run dev
```

Opens at http://localhost:5180

## Add screenshots

| Tab | Folder |
|-----|--------|
| Website | `public/screenshots/website/` |
| Mobile app | `public/screenshots/app/` |
| Flutter web | `public/screenshots/web/` |
| Dashboards | `public/screenshots/dashboards/` |
| 3D Studio | `public/screenshots/studio/` |

Name files `1.png`, `2.png`, … or `screen1.jpg` — they appear automatically.

## APK download

Place your release APK at:

`public/downloads/smartfitao.apk`

The **Download APK** button on the Mobile App tab will serve it.

## Deploy to Vercel

Import this folder as a new Vercel project (root = this directory).
